# Bracket CSV format

The bracket input file should contain first-round matchups.

## Required columns

- `Region` (e.g. East, West, South, Midwest)
- `Round` (use `R64` for first round)
- `GameId` (unique string per game, e.g. `E01`)
- `TeamA`
- `SeedA`
- `TeamB`
- `SeedB`

## Example

```csv
Region,Round,GameId,TeamA,SeedA,TeamB,SeedB
East,R64,E01,Connecticut,1,Stetson,16
East,R64,E02,Florida Atlantic,8,Northwestern,9
```

Only first-round games are required. Later rounds are generated by the simulator.
